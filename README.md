# Store-Data-Analysis-Excel-Dashboard-Project
## Project Objective:
The objective of this project was to perform end-to-end sales data analysis by cleaning, transforming, and visualizing transactional data in Excel, and to develop an interactive dashboard that provides actionable insights on revenue trends, customer behavior, and sales channel performance.

- <a href="https://github.com/darshannagpure5-beep/Store-Data-Analysis-Excel-Dashboard-Project/blob/main/Data%20Analysis%20by%20Excel%20%20Project.xlsx">store data </a>
## Data Model
- <a href="https://github.com/darshannagpure5-beep/Store-Data-Analysis-Excel-Dashboard-Project/blob/main/Dashboard.png">data model </a>

## Project Process
1. Explored and understood raw sales dataset (1000+ records).
2. Cleaned data by correcting formats, removing inconsistencies, and validating values.
3. Standardized Age Group classification using logical formulas.
4. Converted Date column and created Month fields.
5. Built multiple Pivot Tables for trend and category analysis.
6. Designed interactive dashboard using Pivot Charts and Slicers.
7. Applied conditional formatting for better visualization.

## Question KPIs
1. Compare the Sales and Orders using a Single Chart.
2. Who Purchased More – Men or Women in 2022?
3. What Are Different Order Status in 2022?
4. List Top 5 States Contributing to Sales.
5. Relation Between Age and Gender Based on Number of Orders.
6. Which Channel is Contributing to Maximum Sales.

## Conclusion
The analysis of the 2022 store data shows that sales and orders follow a strong seasonal trend, with certain months contributing significantly to overall revenue. Women and customers in the 26–35 age group generated the highest number of orders, making them the key customer segment. Most orders were successfully delivered, indicating good operational efficiency, while a few top states contributed the majority of total sales. Additionally, one major sales channel dominated revenue generation, highlighting the importance of platform-focused strategies for business growth.
